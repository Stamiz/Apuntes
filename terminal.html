TERMINAL - COMANDOS UTILES

  • touch 'nombre del archivo'

    • Crea un archivo con el nombre 'nombre del archivo'

  • sudo pacman -Syu

    • Actualiza el sistema

  • xdg-open .

    • Este comando se utiliza para abrir el Explorador de Archivos en la carpeta actudesde la terminal.

  • htop -> sudo pacman -S htop

    • Administrador de tareas.
